Welcome to Understanding Anxiety, an interactive, beginner-friendly website designed to help users learn about anxiety in a calm, clear, and supportive environment.

This project focuses heavily on simplicity, guided interaction, and a soothing visual experience. Using Bootstrap, custom CSS calm colour themes, and welcoming hero images, the website aims to help users explore anxiety safely and without feeling overwhelmed.

ðŸŽ¯ Project Purpose

Many people who experience anxiety visit educational websites when they are already feeling stressed or unsure about their symptoms. This project supports those users by offering:

Interactive tools that are simple and reassuring

Beginner-friendly explanations of anxiety

A calming interface created with soft colour palettes and gentle layouts

Clear navigation so users never feel lost

Immediate support options for those who need relief fast

The features are designed to balance usability, accessibility, and emotional comfort, ensuring the website remains a safe space for learning.

The website is structured according to a set of carefully prioritised user stories that support both user needs and a calm user experience. These include:

Must-Have Features

Interactive Beginner-Friendly Anxiety Overview
Step-by-step explanations using calming hero images and Bootstrap accordions.

Interactive Coping Tools
Includes breathing animations and grounding tools styled with a gentle colour palette.

Interactive Symptom Checker
A beginner-friendly, card-based flow with progress indicators.

Bootstrap Navigation Bar (Calming + Beginner Friendly)
A simple top nav with clear labels, soft colours, and mobile-friendly collapse.

Should-Have Features

Choose Your Path Learning Journey
Users explore topics using interactive cards and guided steps.

Interactive Treatment Explorer
Bootstrap tabs/pills that explain treatment options in simple, approachable terms.

Could-Have Features

Interactive Resource Finder
A supportive, dropdown-based tool for finding help or crisis resources.

These user stories ensure the project stays aligned with the needs of anxious users, providing a structured and supportive learning experience.
